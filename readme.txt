USING THE enrolltranscript Block
-----------------------------

1. Unzip the archive and read this file  ;-)

2. Copy the "enrolltranscript" directory into the block folder.

3. Login as the admin and click on the notifications.

4. Click on the turnediting on and add the "Enrollments And Transcripts" block .

5. You will see the "Enrollments And Transcripts" on the page where you added.

Major features
--------------
You can see all the current course enrollments of the user along with the date of enrollment. Once the user completes the course it is moved to the Transcript.
In the transcript you will find the date of completion and the list of activities having the course completion criteria.
